# Guitar Fretboard Interval Trainer

ギター指板上でルート音からの度数（インターバル）の絶対位置を、**耳と手を直結させながら**体に染み込ませる自習用PWAアプリ。

「音が鳴ったら指が動く」感覚を養うため、理論の暗記ではなく反復タップによる身体化を重視した設計です。

---

## コンセプト

- **2タップ方式** — まず基準音（Root）をタップして耳と指板の位置を結びつけ、次に度数音をタップして答える
- **音を聴いてから探す** — 問題音を毎回再生し、音名を見る前に音で感じる
- **指板全体を活用** — 高レベルでは基準音の弦・オクターブもランダム化し、特定ポジションへの依存を排除

---

## 機能

### インターバル編（v1.0 実装済み）

| LV | 出題音 | 基準音 | 判定弦 |
|---|---|---|---|
| LV.1 | R・3rd・5th | C 固定 | 低音3弦（6〜4弦） |
| LV.2 | +2nd・4th | C 固定 | 低音4弦（6〜3弦） |
| LV.3 | +m3・m7 | E/A/D/G/C/B ランダム | 低音5弦（6〜2弦） |
| LV.4 | +6th・#5 | 12音フルランダム、6〜4弦×0/1oct | 全弦 |
| LV.Max | 全12音 | 12音フルランダム、6〜4弦×0/1oct | 全弦 |

- 練習モード（タイムなし、好きなだけ反復）
- 10問セット制、タイムランキング上位20件をローカル保存
- 正解時に 0.5 秒後自動で次問、不正解は色フィードバックのみで同ポジション待機
- ホーム画面の各LV横の🏆ボタンから、プレイせずにそのLVのランキングだけを確認可能
- 詰まった時のヒント表示（5〜7秒無反応でフェードイン表示、スコア・ランキングへの影響なし、ハンバーガーメニューでON/OFF切替可、デフォルトON、LV.Maxのみ対象外）

### コードトーン編（v1.5 Stage 0 実装済み）

コード名を聴いて、構成音を指板上ですべてタップする形式。

- **Stage 0**（現在）: 練習モードのみ（タイマーなし・無限ループ）。ルート C 固定、メジャー / マイナートライアドのみ、低音3弦（6〜4弦）が対象
- 判定は「未クリア集合」方式: 構成音（例: R・m3・5th）をタップ順不同で探し、見つけた度数を集合から消す。常時「R ✓　m3 ✓　5th（未）」のようなテキストで進捗表示
- コードは複数音を同時発音（ストローク再生）
- Stage 1以降で7th・テンション・出題度数の部分集合・対象弦の可変設定・出題順序制約を段階的に追加予定

## 指板表示の仕様

- **表示範囲**: 基準音を中心とした6フレット幅（0F〜17F 内で動的に決定）
- **開放弦**: ナット左側に専用ゾーンを表示、タップ可能
- **ポジションマーク**: 3・5・7・9・12・15・17F（実際のギターに準拠）
- **出題バリデーション**: 「18F以上にしか正解がない」問題は最大30回のリトライで自動排除

## PWA 対応

- ホーム画面に追加してオフライン動作
- `manifest.json` で横持ち（landscape）固定
- 縦持ちアクセス時は「横向きにしてください」オーバーレイを表示

## ローカルでの起動方法

iOS Safari で Web Audio API を使うには **HTTPS** が必要です。

### 1. 証明書の生成（初回のみ）

```bash
brew install mkcert nss
mkcert -install
mkcert 192.168.x.x localhost 127.0.0.1
# 生成された .pem ファイルを cert.pem / key.pem にリネーム
```

### 2. サーバー起動

```bash
node server.js
```

iPad/iPhone から `https://192.168.x.x:3443` でアクセス（同一 Wi-Fi 必須）。

## v1.5 以降に保留している機能

- **コードトーン編 Stage 1以降** — 7th・テンション、出題度数の部分集合、対象弦の可変設定、出題順序制約（順序固定⇄順不同の段階的導入）
- **指板幅の可変設定** — 表示フレット幅を 5・6・7 から選択
- **ハプティクスフィードバック** — 正解・不正解を振動で区別（iOS PWA 限定）
- **CAGED システム表示** — 正解後に該当ポジションのケイジドフォームをハイライト
- **カスタム出題設定** — 出題度数・判定弦をユーザーが個別に指定
- **音源の追加** — サンプラー音源（実際のギター音）への対応
