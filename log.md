# Changelog

## v1.1.0 — 2026-06-30

### 機能拡張・バグ修正
- **17F対応**: 表示・タップ判定・出題バリデーションを 0F〜17F に統一
  - `MAX_FRET = 17`（旧: 16）
  - `calcDisplayRange` の上限を `MAX_FRET-1 = 16` に修正（最大表示ゾーン = 17F、18F 表示バグを解消）
  - ポジションマークに 17F を追加
- **基準音バリエーション拡張（LV.4 / LV.Max）**: `rootStrings: [0,1,2]`・`rootOctaves: [0,1]` で 6〜4弦 × 0/1 オクターブからランダム選択
- **出題バリデーション**: `_hasValidAnswer()` + 最大 30 回リトライで「18F 以上にしか正解がない」問題を排除
- **開放弦対応**
  - ナット左側に 0F 専用ゾーンを描画、タップ可能
  - フレット番号を `0, 1, 2, 3...` 形式で表示
  - 開放弦を回答対象から除外できる設定をハンバーガーメニューに追加
- **波形デフォルト変更**: `triangle` → `square`
- **2タップ方式**: フェーズ1（Root タップ）→ フェーズ2（度数音タップ）の順を強制
- **縦持ち対応**: CSS rotate 削除、「横向きにしてください」オーバーレイ方式に変更
- **PWA**: Service Worker を `fretboard-v8` に更新

## v1.0.0 — 2026-06-29

### 初期実装
- インターバル編: LV.1〜LV.Max + 練習モード
- 指板Canvas描画（横持ち、左ヘッド/右ブリッジ、6フレット幅動的表示）
- Web Audio API OscillatorNode音再生（sine/triangle/sawtooth/square切り替え）
- ルートからの半音数によるピッチクラス判定 + テンション判定
- スコアランキング（モード×LVごとにlocalStorageに上位20件保存）
- PWA対応（manifest.json + Service Worker）
