このアプリについて

ギターの指板上で「耳で聴いた度数（インターバル）」の位置を直接タップして覚えるトレーニングPWAです。

目指すのは 「音が鳴ったら指が動く」 状態。
コードネームや音楽理論を頭で処理するより先に、
耳と手が直結 するまで反復することが目的です。

問題のたびにルート音と度数音を実際に再生し、音名を見る前にまず「聴く」ことを意識してください。指板上のポジションは画面をタップするだけで確認できます。

ホーム画面への追加（推奨）

ホーム画面に追加してPWAとしてインストールすると、横向きが自動でロックされます。Safari ブラウザのまま使う場合は画面を手動で横向きにしてください。

iOS / iPadOS（Safari）

  1. Safari でこのアプリのURLを開く
  2. 画面下部の 共有ボタン（□に↑のアイコン）をタップ
  3. 「ホーム画面に追加」 を選択
  4. 名前を確認して 「追加」 をタップ
  5. ホーム画面のアイコンから起動すると、横向きが自動でロックされた状態でアプリが開きます

Chrome on iOS ではホーム画面への追加ができません。必ず Safari を使用してください。

縦向きでアクセスした場合は「横向きにしてください」と表示されます。端末を横向きにすると指板が表示されます。

遊び方

ゲームの流れ

  1. 音を聴く
    問題が始まると、ルート音と度数音が順番に再生されます。もう一度聴きたいときは 「もう一度」ボタン をタップしてください。
  2. ① Root をタップ
    ヘッダーに音名（例: 「C」）が大きく表示されます。指板上でその音をタップしてください。正しい位置をタップするとオレンジの印が付いて次のフェーズへ進みます。
  3. ② 度数音をタップ
    ヘッダーに度数名（例: 「5th」）が表示されます。ルートから見てその度数にあたる音を指板上でタップしてください。
  4. 正解 → 次の問題へ
    正解すると緑のマーカーが表示され、0.5秒後に自動で次の問題へ進みます。不正解は赤のマーカーが表示され、同じポジションで待機します（何度でもタップし直せます）。

タップに詰まって5〜7秒経つと、候補ポジションに半透明の丸がフェードイン表示されます（ヒント機能）。ヒントを使ってもスコアやランキングには一切影響しません。ハンバーガーメニューからON/OFFを切り替えられます（デフォルトはON）。LV.Max ではヒントは表示されません。

ヘッダーの見方

  1. 問題番号（例: 3 / 10） — 現在が何問目かを示します
  2. タイマー — 問題画面に入った瞬間から止まらずに動き続ける経過時間です（出題音の再生中や問題の切り替わりも含みます）。ランキングに保存されるタイムはこの表示とは別に計算されており、各問題で出題音の再生が終わってから正解するまでの時間だけを10問分合計したものです。「もう一度」で聞き直しても、その分の時間はスコアにきちんと加算されます
  3. 音名 / 度数名 — フェーズに合わせて現在タップすべき情報を大きく表示

10問終わるとリザルト画面へ進み、正解数とタイム、過去のランキングが表示されます。

正解判定のルール

このアプリ独自の判定ルールがあります。理解しておくと答えやすくなります。

コードトーン（現在の全LV対象）

R・2nd・m3・3rd・4th・b5・5th・#5・6th・m7・7th は
どのオクターブでも正解 です。
たとえば「5th を探せ」という問題で、ルートより低い位置に G があっても正解になります。

ピッチクラス（音名）が一致していれば、ルートより上でも下でも正解として扱います。指板全体から探してください。

テンション（9th・#9・b9・11th・#11・13th など）

テンションは ルートから1オクターブ以上高い位置 にあるもののみ正解です。
同じ音名でもルートより低い位置や同オクターブ内の音は不正解になります。

テンションはルートに対して「上に積み重なった音」という意味合いがあるため、オクターブが重要になります。現バージョン（LV.1〜LV.Max）ではテンションは出題されません。

判定弦が限られている場合、正解が見つからないことはある？

出題時に「指定された判定弦の表示範囲内に正解が存在するか」を自動チェックしています。答えが見つからない組み合わせは最大30回再生成されるため、タップ可能な範囲に必ず正解が存在します。

タップしても反応しない位置がある

暗く塗りつぶされている弦（マスクされた弦）はそのレベルの対象外です。暖色でハイライトされている弦だけがタップ可能な判定弦です。また、ハンバーガーメニューの「開放弦」ボタンが「除外」表示になっている場合は、0フレット（開放弦）の音が正解でもタップが無効になります。

難易度レベル

ホーム画面の「インターバル編」から難易度を選択します。LV が上がるほど出題音・基準音・判定弦が増えます。

モード 出題される度数 基準音 タップ可能な弦
練習 R・3rd・5th C 固定 6・5・4 弦
LV.1 R・3rd・5th C 固定 6・5・4 弦
LV.2 +2nd・4th C 固定 6・5・4・3 弦
LV.3 +m3・m7 E/A/D/G/C/B ランダム 6・5・4・3・2 弦
LV.4 +6th・#5 12音フルランダム
6〜4 弦×0/1 オクターブ		 全弦
LV.Max 全12音 12音フルランダム
6〜4 弦×0/1 オクターブ		 全弦

練習モードはタイムの計測とランキング保存がなく、何度でも繰り返せます。初めての方はまず練習モードで操作感を掴んでください。

LV.4 / LV.Max では基準音が 6・5・4 弦のいずれかにランダムで配置され、さらに 1 オクターブ上のポジションも候補になります。高フレット寄りの問題が出た場合は、指板が自動でスクロールします。

どのレベルから始めればいい？

まず「練習モード」でタップ操作に慣れてから LV.1 へ進むことを推奨します。LV.1〜LV.2 は基準音が C 固定なので、C メジャースケールや C コードのポジションを把握しているギタリストなら比較的スムーズに取り組めます。

スコアはどこに保存される？

スコア（タイム）はレベルごとに端末のローカルストレージに保存され、10問を解き終えるとリザルト画面でランキングとして確認できます。iOS Safari のプライベートブラウズモードでは保存されません。

練習モードにはスコア記録・リザルト画面がありません。タイム計測なしで何度でも反復できるモードのため、10問という区切りがなく無制限に出題が続きます。タイムを記録したい場合は LV.1 以上を選んでください。
